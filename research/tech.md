# Tech/SaaS Industry Research: Nexus Labs

## 1. Color Theory & Examples
- **Primary Palette**: Cyber Purple (#6A0DAD), Electric Teal (#008080), and Dark Mode Navy (#0B0E14).
- **Vibe**: Innovative, futuristic, sleek, clean.
- **Example Sites**: Stripe, Linear, Vercel.
- **Psychology**: Purples and teals represent the "future" and "digital" space.

## 2. Animation & Style Inspiration
- **Key Animations**: 
  - Floating UI elements (mockups).
  - Gradient borders that animate on hover.
  - Smooth "flow" diagrams using GSAP DrawSVG.
  - Micro-interactions on buttons (glow, scale).
- **Awwwards Inspiration**: "Stripe" (gradient work), "Framer" (interaction design).

## 3. Media & Text Strategy
- **Media**: Clean software mockups, abstract 3D renders (blobs, glassmorphism).
- **Typography**: Ultra-modern Sans-Serif (e.g., Geist or San Francisco).
- **Tone**: Efficient, visionary, simple.
- **Sample Copy**: "The infrastructure for what's next. Deploy in seconds, scale for eternity."
